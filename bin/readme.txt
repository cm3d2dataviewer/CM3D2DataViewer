■概要
CM3D2のGameData内のデータを解析、表示するツールです(一部編集機能もあり)

■必要環境
.NET Framework 3.5

■事前準備、注意点
　・CM3D2Toolを利用してGameData以下の.arcファイルをディレクトリに展開しておいて
　　ください。
　・使用前に事前にゲームデータのバックアップを取ってください。

■使い方
●環境設定
　・CM3D2DataViewer.exeを起動する。
　・事前に展開しておいたGameDataフォルダを表示されたウィンドウに
　　ドラッグ＆ドロップします。
　　ディレクトリ内を全て読み込むので、処理に若干時間がかかります。
　・(任意で)ReiPatcher.exeをウィンドウにドラッグ＆ドロップすると、上部ツール
　　バーからゲームが起動できるようになります。

　上記の設定は保存されるので、次回以降の起動時には不要です。

●menu内容の閲覧
　・ウィンドウ左側に.menuファイルをカテゴリごとに分類したツリーが表示
　　されます。
　・選択すると.menuファイルの内容が表示されます。(画面中央)
　・関連するファイルがボタンで表示され、クリックすることで該当の
　　ファイルを右側タブ内で開きます。
　・「関連リソース」タブの「関連リソースの解析」ボタンで、menuと
　　スクリプトに記載されているファイルを順次読み込み、データの関係性を
　　示すツリーが表示します。

●アイテム(menu)の複製
　既存のアイテムのファイル名を変更して複製を作成します。
　・.menuを開き、「コピー」タブを選択する
　・「ファイル抽出」ボタンを押す
　・ファイル名置換のテキストを修正する。(リストにコピー前後のファイル名が表示される)
　・コピーボタンを押す

●ファイルの編集
　・menu,mateファイルはテキストボックス内を修正後、「保存」ボタンで
　　修正内容をファイルに反映できます。

●テクスチャのインポート、エクスポート
　・texの画面では、画像ファイルの「インポート」「エクスポート」が
　　可能です。

●モデルのインポート、エクスポート
　modelの画面では、モデルデータの「インポート」「エクスポート」が可能です。
　OBJ形式でのみエクスポート＆インポートができます(その他はエクスポートのみ)
　エクスポート時にはモーフィングなどを含まない基本形状のみを出力し、
　インポート時は参照モデルファイルからモーフィングやスキンの設定を自動的に取得して、
　modelファイルを作成します。

　(モデルをインポート、エクスポートできるツールは他にもあるので、
　　このツールではあくまでおまけ的な機能です)

●ドラッグ＆ドロップでのファイル閲覧
　エクスプローラーなどから、直接ファイルをドラッグ＆ドロップすることでも、
　ファイルの閲覧ができます。(.mate.menu .model .texに対応)
